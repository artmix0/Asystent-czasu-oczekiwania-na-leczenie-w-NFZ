# Asystent czasu oczekiwania na leczenie w NFZ
Nowoczesna aplikacja webowa, która pełni rolę osobistego doradcy w gąszczu publicznej służby zdrowia. Zamiast ręcznego przeszukiwania skomplikowanych baz danych, użytkownik komunikuje się z systemem w języku naturalnym, aby błyskawicznie znaleźć najkrótszą drogę do specjalisty lub badania.
